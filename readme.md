# Expense Analysis Web App

This web application allows users to **log in**, **upload their monthly expense Excel file**, and view a **Chart.js graph** of their daily spending.  
An analysis panel shows the **highest spending day**, **lowest spending day**, and **average daily expense**.

## Features
- User login and registration  
- Upload daily expense Excel file  
- Chart.js visualization of daily expenses  
- Summary analysis: highest, lowest, and average spending  

## How It Works
1. Log in to your account  
2. Upload your monthly expense Excel file  
3. View the daily expense chart  
4. Check the analysis panel for quick insights  

## Excel Requirements
Your Excel file should include (in specific order, column name may differ):
- **Date**
- **Product name / ID**
- **Unit Price**
- **Quantity**
