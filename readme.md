Expense Analysis Web App

This web application allows users to log in, upload their monthly expense Excel file, and view a Chart.js graph of their daily spending.
An analysis panel shows the highest spending day, lowest spending day, and average daily expense.

Features

User login and registration

Upload daily expense Excel file

Chart.js visualization of spending for the month

Summary analysis: highest, lowest, and average expense

How It Works

Log in to your account

Upload your monthly expense Excel file

View the daily spending chart

See quick insights from the analysis panel

Requirements

The Excel file should include:

Date

Daily expense amount
